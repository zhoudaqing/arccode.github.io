阿里云CentOS搭建MEAN

发表于 2014-09-21   |  

购买主机

看个人爱好，本人购买阿里云主机，最低配置55元/月

安装相关模块

  1. 在阿里云管理控制台启动购买的云主机
  2. 使用zoc6连接服务器

  3. 更新软件 yum -y update

  4. 安装开发工具 yum -y groupinstall "Development Tools"

  5. 下载Node源码
    • cd /usr/src
    • wget http://nodejs.org/dist/v0.10.32/node-v0.10.32.tar.gz
  6. 安装Node
    • 解压node-v0.10.32.tar.gz
    • cd node-v0.10.32
    • ./configure
    • make
    • make install
    • *注：编译后的二进制文件会放置于系统路径下/usr/local/bin/node
