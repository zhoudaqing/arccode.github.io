Mac下安装memcached

发表于 2014-08-10   |  

安装libevent(memcached的依赖库)

  • 下载libevent源码
  • 解压tar -xzf libevent-2.0.21-stable.tar.gz
  • cd libevent-2.0.21-stable
  • sudo ./configure --prefix=/usr
  • sudo make
  • sudo make install

安装memcached

  • 下载memcached源码
  • 解压memcached-1.4.20
  • cd memcached-1.4.20
  • sudo ./configure --prefix=/usr
  • sudo make
  • sudo make install

Svn基本使用

发表于 2014-07-22   |  

基本使用

图标说明

  • 灰色向右箭头:本地修改过
  • 蓝色向左箭头:SVN上修改过
  • 灰色向右且中间有个加号的箭头:本地比SVN上多出的文件
  • 蓝色向左且中间有个加号的箭头:SVN上比本地多出的文件
  • 灰色向右且中间有个减号的箭头:本地删除了,而SVN上未删除的文件
  • 蓝色向左且中间有个减号的箭头:SVN上删除了,而本地未删除的文件
  • 红色双向箭头:SVN上修改过,本地也修改过的文件
